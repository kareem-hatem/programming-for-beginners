Making a loader

Step 1: add html

<div class="loader"></div>

Step 2: add css


      .loader{
          border: 16px solid #eee /*light grey*/;
          border-top: 16px solid #3498db; /* Blue */
          border-radius: 50%;
          width: 120px;
          height: 120px;
          animation: spin 2s linear infinite;
          }
      @keyframes spin{
          0% { transform: rotate(0deg); }
          100% { transform: rotate(360deg); }
      }
