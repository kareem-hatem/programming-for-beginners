Text decoration


Bold and Strong


Both bold and strong make the text inside them thicker like this 

      
        <p><b>content</b></p>

Interactive background below



Italic and Em


Both italic and em rotate the text inside them a little like this 

      
        <p><em>content</em></p>

Interactive background below



Underline


Underline underlines the text inside them like this

      
        <p><u>content</u></p>

Interactive background below


Strikethrough


Strikethrough makes a line throught the text inside them like this

      
        <s>Text to display</s>
