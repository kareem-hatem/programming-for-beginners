Making a fixed sidebar

Step 1: add html


<div class="sidebar">
  <h2 class="sidebar-heading">Hello</h2>
  <a href="url or file">Text to display</a>
  <a href="url or file">Text to display</a>
  <a href="url or file">Text to display</a>
</div>

Step 2: add css
Note: you put your enitre website in a div with a class called 'main' 


.sidebar{
  height: 100%; /* Full-height: remove this if you want "auto" height */
  width: 200px; /* Set the width of the sidebar */
  position: fixed; /* Fixed Sidebar (stay in place on scroll) */
  z-index: 1; /* Stay on top */
  top: 0; /* Stay at the top */
  left: 0;
  background-color: #111; /* Black */
  overflow-x: hidden; /* Disable horizontal scroll */
  padding-top: 20px;
}
.sidebar a {
  padding: 6px 8px 6px 16px;
  text-decoration: none;
  font-size: 19px;
  color: #818181;
  display: block;
  word-spacing: normal;
  margin-top: 8px;
}
.main {
  margin-left: 200px; /* Same as the width of the sidebar */
  padding: 0px 10px;
}
.sidebar-heading{
  padding: 6px 8px 6px 16px;
  text-decoration: none;
  font-size: 19px;
  color: #818181;
  display: block;
  word-spacing: normal;
}
.sidebar a:hover {
  color: #f1f1f1;
}
